                                                  Краткое описание
      Для чего предназначено приложение
AirGuard KZ — мобильное приложение для мониторинга качества воздуха в реальном времени, прогноза погоды и климатических рекомендаций.
Приложение показывает текущий индекс качества воздуха (AQI), уровень основных загрязнителей (PM2.5, PM10, NO2), прогноз на пару дней вперёд, карту станций мониторинга и даёт простые рекомендации: «Наденьте маску», «Не открывайте окна», «Отложите пробежку», «Подходит для прогулки с детьми».
      Какую проблему решает
В промышленных городах Казахстана (Караганда, Темиртау, Алматы, Павлодар и др.) качество воздуха часто плохое из-за угольной энергетики, шахт и автотранспорта, но люди не всегда знают актуальные данные и риски для здоровья. Официальные источники неудобны, а международные приложения не всегда точно покрывают Казахстан. AirGuard KZ решает это, собирая данные из доступных API (aqicn.org, открытым казахстанским источникам, если появятся), показывая их просто и на русском/казахском, и помогая людям принимать ежедневные решения для защиты здоровья.
